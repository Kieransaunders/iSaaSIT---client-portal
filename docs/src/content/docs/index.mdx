## What is iSaaSIT?

iSaaSIT is an open-source SaaS starter kit designed for agencies and freelancers who need a repeatable client project template.

<CardGrid stagger>
  <Card title="Multi-Tenant Architecture" icon="cloud">
    Manage multiple client companies with proper data isolation using Organizations and Customers.
  </Card>
  <Card title="Role-Based Access Control" icon="lock">
    Built-in roles (Admin, Staff, Client) with scoped permissions for secure data access.
  </Card>
  <Card title="Authentication Ready" icon="approve-check">
    Pre-configured WorkOS AuthKit integration with JWT validation and protected routes.
  </Card>
  <Card title="Modern Stack" icon="rocket">
    Built with TanStack Start, Convex, React 19, Tailwind CSS v4, and TypeScript.
  </Card>
  <Card title="Billing Integration" icon="credit-card">
    Planned Lemon Squeezy integration for usage-based billing with plan caps.
  </Card>
  <Card title="Client Portal" icon="users">
    Self-service portal for clients to manage their own data and users.
  </Card>
</CardGrid>
